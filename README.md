# 🍽️ Happy_Bites

Welcome to **Happy_Bites**, a fictional restaurant website built using HTML, CSS, and Bootstrap. This project showcases a responsive and visually appealing design for a modern restaurant.

- Bootstrap Grid
- Navbar
- Hero section 
- About Us section
- Menu section
- Contact section.

- CSS
- Bootstrap
